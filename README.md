# URL Shortener (Full Stack SaaS App)

A full-stack URL shortener application with authentication, analytics, and JWT-based cookie login system.  
Deployed with **Frontend on Vercel** and **Backend on Render**. 

---

## Features

- User Authentication (Register/Login/Logout)
- JWT stored in cookies
- Create short URLs
- Track URL clicks
- User dashboard for all links
- Real-time API integration
- Fully deployed full-stack app

---

## Tech Stack

**Frontend:**
- React (Vite)
- Redux Toolkit
- Axios
- TanStack Router

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Cookies 

**Deployment:**
- Vercel (Frontend)
- Render (Backend)

---

## Architecture

Frontend (Vercel)  
→ API Calls (Axios)  
→ Backend (Render)  
→ MongoDB Database  

Authentication handled using **JWT + HTTP-only cookies**.
